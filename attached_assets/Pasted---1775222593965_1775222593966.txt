ПУНКТ 1: Концентрации перекиси водорода и плотности

Описание

Добавить возможность работы с растворами пероксида водорода (H₂O₂). Пользователь может:

· Ввести плотность раствора → получить массовую концентрацию (%)
· Ввести массовую концентрацию (%) → получить плотность
· Пересчитать между % масс., молярной концентрацией (моль/л), количеством вещества (моль) для заданного объема.

Таблица зависимости плотности от концентрации (для H₂O₂)

% масс. H₂O₂ Плотность (г/мл, 20°C)
3% 1.009
6% 1.019
10% 1.035
30% 1.110
35% 1.130
50% 1.195
70% 1.290
85% 1.360
98% 1.420

(в коде должна быть интерполяция для промежуточных значений)

Алгоритм

```
Функция get_h2o2_concentration():
    Вывести меню:
    1 - Рассчитать % по плотности
    2 - Рассчитать плотность по %
    3 - Пересчет количества вещества
    
    Если выбран 1:
        Ввод: плотность (г/мл)
        Поиск в таблице (линейная интерполяция между ближайшими значениями)
        Вывод: % масс. H₂O₂
    
    Если выбран 2:
        Ввод: % масс. H₂O₂
        Поиск плотности в таблице (интерполяция)
        Вывод: плотность (г/мл)
    
    Если выбран 3:
        Ввод: % масс. H₂O₂
        Ввод: объем раствора (мл)
        Расчет:
            m(H₂O₂) = V_раствора * плотность * (% / 100)
            n(H₂O₂) = m(H₂O₂) / 34.0147 (молярная масса H₂O₂)
            C_molar = n(H₂O₂) / (V_раствора / 1000)  # моль/л
        Вывод: количество вещества (моль), молярная концентрация (моль/л)
```

Колонка Тип Описание
synthesis_id INTEGER Внешний ключ
reagent_name TEXT Название
